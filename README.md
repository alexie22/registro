# Sistema de Registro de Alumnos

## Descripción
El Sistema de Registro de Alumnos es una aplicación diseñada para gestionar la información académica de los estudiantes de manera organizada. Permite registrar, consultar, actualizar y eliminar datos de los alumnos, facilitando la administración dentro de una institución educativa.

## Objetivos
- Mantener un registro ordenado de los estudiantes.
- Facilitar la gestión de información académica.
- Reducir errores en el manejo manual de datos.
- Optimizar el acceso y la actualización de la información.

## Funcionalidades
- Registro de nuevos alumnos.
- Consulta de información de estudiantes.
- Actualización de datos de alumnos.
- Eliminación de registros.
- Listado completo de estudiantes registrados.
